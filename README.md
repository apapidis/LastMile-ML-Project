# Last Mile Delivery Duration Prediction

## Project Description

This project explores how to predict delivery route duration using the Amazon Last Mile Routing Research Challenge dataset, which includes data for 6,112 delivery routes.

The goal is to estimate how long a delivery route will take before it actually happens, using machine learning models.

## Methodology

The workflow followed three main steps:

1. Data cleaning and preprocessing
2. Exploratory Data Analysis (EDA)
3. Training and evaluation of machine learning models

Models used:

* Linear Regression
* XGBoost
* Random Forest
* Gradient Boosting

## Results

The best-performing models achieved an average prediction error (MAE) of around 18 minutes, while the average route duration is between 6–8 hours.

## Key Insight

One of the main takeaways is how important delivery service time is.
It can vary a lot depending on real-world conditions such as:

* parking availability
* building type
* delivery environment

This suggests that having more detailed delivery-related data could further improve the model’s accuracy.

## Dataset

Amazon Last Mile Routing Research Challenge Dataset (2021)
